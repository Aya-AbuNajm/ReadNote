# Styling text

1-Bold text
~~~
ex: use ** text **
~~~
**aya abu najm**

2-Italic text
~~~
ex:  use *text*
~~~
 *aya abu najm* 
 
> 2-Hidden text
~~~
ex:  use >  text
~~~
> aya abu najm 

3-Quoting
 you can use ~~~ to text
